# HTML

### HTML
`HyperText Markup Language`

### 기본 문법

시작태그와 닫히는 태그가 있음.

#### 주석 : ```<!-- 주석내용 -->```
```html
<!-- 안녕 나는 주석이야 -->
```

#### 제목 : ```<h1> </h1>```
```html
<h1>START!</h1>
<!-- 숫자가 바뀌면 크기가 바뀜 -->
```

#### 강조하기 : ```<strong> </strong>```
```html
Hello This is my First <strong>HTML Programming</strong>.
```
